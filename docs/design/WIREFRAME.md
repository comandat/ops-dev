## 🎯 Section Breakdown

### 1. Navigation
- **Layout**: Fixed top bar, sticky on scroll
- **Elements**: Logo (left), Nav links (center), Login + CTA (right)
- **Mobile**: Hamburger menu, logo + login visible
- **Height**: 64px desktop, 56px mobile

### 2. Hero Section
- **Layout**: Centered content, max-width 800px
- **Elements**: H1 headline, subheadline, 2 CTAs, trust badges
- **Visual flow**: Headline → Subhead → Primary CTA → Secondary CTA → Trust indicators
- **Padding**: 120px top/bottom desktop, 80px mobile

### 3. Stats Bar
- **Layout**: 2-4 stats in a row, equal width
- **Elements**: Large number + label
- **Background**: Slightly lighter than base (#121217)
- **Padding**: 48px top/bottom

### 4. Features Section
- **Layout**: 4-column grid desktop, 1-column mobile
- **Elements**: Section title, 4 feature cards (icon + title + description)
- **Card structure**: Icon (48px) → Title (20px) → Description (16px)
- **Gap**: 24px between cards

### 5. Pricing Section
- **Layout**: 4-column grid desktop, stacked mobile
- **Elements**: Section title, 4 pricing cards
- **Card structure**: Plan name → Price → Features list → CTA
- **Highlight**: Starter plan with border + "Popular" badge
- **Gap**: 24px between cards

### 6. Final CTA
- **Layout**: Centered, max-width 600px
- **Elements**: Headline, subheadline (trial info), CTA button, trust badges
- **Background**: Gradient accent or card style
- **Padding**: 96px top/bottom

### 7. Footer
- **Layout**: 4-column grid + bottom bar
- **Elements**: Logo + description, link columns, contact, legal
- **Padding**: 64px top, 32px bottom

